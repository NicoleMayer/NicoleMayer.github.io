生活流水账 ｜ alfred | vuepress

  1. 继上次找到了OCR网站，又发现了神器，alfred直接一键OCR，不要太方便了！下载地址在这，主要是利用了tesseract这个命令:

    Tesseract has unicode (UTF-8) support, and can recognize more than 100 languages “out of the box”.

    可见这个命令有多么强大，最新版添加了基于LSTM训练的模型。但是刚刚那个网址提供的脚本有点小问题，必须提供语言否则识别会出错。稍微改一改脚本加个if-else判断就好了。电脑原有的tesseract没有中文的训练集，去github仓库地址下载好拷贝到/usr/local/Cellar/tesseract/4.0.0_1/share/tessdata/文件夹下就行了，氮素这个中文效果不敢恭维…所以我改了改脚本，如果中文效果不好就打开OCR的网站。

  2. mac使用 Command + Shift + . 就可以在finder下面显示隐藏文件。

  3. 终于配置好了 vuepress，使用了这个开发者的vuepress-bar自动生成侧边栏，有点小问题，老是会显示路径和后缀而不是单纯的文件名，之后再慢慢想办法解决。对中文的适配不是很好，我搞不太懂为什么用拼音的插件，直接中文不好嘛.. 下面摘录几个用到的命令：

    
yarn add -D vuepress

npm install -D boboidream/vuepress-bar

npm i vuepress-plugin-permalink-pinyin --save-dev

yarn docs:build

yarn dev

    还需要注意的是文件夹层级结构，第一层在顶部导航栏，第二层在侧边栏，可以根据数字大小排序。此外，每一个系列必须有README，否则网页就会渲染不出来。

  4. 晚上和室友们一起恰火锅，一个室友现学的韩式火锅，超赞的！
