overview

TROIKA 三步走： 1. signal decomposition 去除原始PPG信号中的MA，并且稀疏其频谱 2. sparse signal re-construction (ssr) 计算高分辨率PPG信号的频谱 3. spectral peak tracking 寻找心率对应的光谱峰（spectral peak）

details

T秒的滑动窗口，以S的步长增加（S <= T/2，T = 8， S = 2） 0.4～5Hz之间的带通滤波器

A signal decomposition

方法有：singular spectrum analysis (SSA) / single-channel independent component analysis 等等 他们选择了 SSA，包括四个步骤： Embedding, Singular Value Decomposition (SVD), Grouping, and Reconstruction.

B Temporal Difference Operation

对得到的信号做二阶差分

C Sparse Signal Reconstruction

$$ y = Φ x + v $$

y is an observed signal of the size M × 1

x is an unknown solution vector which is assumed to be sparse or compressive

v is an unknown noise vector

The goal is to find the sparsest vector x based on y and Φ.
